# 理論文書

## 概要

本研究は、専門用語を「概念の論理的圧縮」として捉え、統計的学習と論理的検証を統合した理論的枠組みを提案します。

> **中心仮説**: 専門用語とは、複数の概念・前提・制約を、論理構造を保ったまま短い記号列へ圧縮した表現である。

この視点は、自然言語処理における専門用語の扱いに新たな理論的枠組みを提供し、特に大規模言語モデル（LLM）における知識表現と推論の本質的理解に寄与します。

## 用語統一

本文書群では、以下の用語を統一的に使用します：

- **抽象化**: compression（概念の圧縮）
- **展開**: decompression / expansion（圧縮された概念の展開）
- **制約検証**: validation（論理的制約の検証）
- **専門用語**: constraint-bearing compressed concept（制約を保持する圧縮された概念）
- **高位語**: 高度に圧縮された概念表現
- **低位概念**: 展開された基底概念列
